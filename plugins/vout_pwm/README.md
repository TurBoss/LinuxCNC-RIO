# Plugin: vout_pwm

Variable-Output for PWM-Signals with optional DIR pin

100kHz PWM with DIR-Pin

```
{
    "type": "pwm",
    "frequency": "100000",
    "pin": "T3",
    "dir": "R4"
},
```

setup for rcservo's (1-2ms @ 50Hz)

```
{
    "type": "rcservo",
    "frequency": "50",
    "pin": "M5"
},
```
